5. **代码**  
     - 多行代码使用键盘左上角的小点（英文字符、3个小点）
    ```
     def rosenbrock_function(x):
    """
    Rosenbrock 函数
    输入:
        x: 由n个变量组成的数组 (list 或 numpy 数组)
    输出:
        Rosenbrock 函数的值 (float)
    """
    # 计算Rosenbrock函数值
    return sum(100 * (x[i + 1] - x[i]**2)**2 + (x[i] - 1)**2 for i in range(len(x) - 1))
    ```
